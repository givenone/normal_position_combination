SPQR/Doc directory for SuiteSparseQR

Please read the spqr_user_guide.pdf first.

For additional documentation of the MATLAB user interface, type:

    help spqr
    help spqr_solve
    help spqr_qmult
    help spqr_singletons

in the MATLAB Command Window.
